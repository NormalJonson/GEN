# GENTOO INSTALL SCRIPT ;)
A Bash-based automate Gentoo Linux Install Script. It can also be updated for any system user desires.

## Features
- Disk partitioning and formatting
- Stage3 setup (You should select the latest stage)
- Chroot automation
- Kernel and bootloader installation
- Basic system configuration

## Usage
- ⚠️ This script will erase all data which in this case is `/dev/sda`.
- curl the repo in the LiveCD.

<pre> chmod +x Gentoo_installV1-5.sh </pre>
<pre> ./Gentoo_installV1-5.sh </pre>

## Notes
- Follow the prompts during installation otherwise it might not work properly.
- Best used in a VM or test environment.
- Created to simplify the manual Gentoo installation process.
